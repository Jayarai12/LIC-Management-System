# LIC Management System (C++)

This is a console-based LIC (Life Insurance Corporation) policy management system made using C++ and file handling.  
It allows the user to add, view, search, update and delete LIC policy records which are stored in a text file.

## Features

- Add new policy
- View all policies
- Search policy by policy number
- Update existing policy
- Delete policy
- Data stored permanently in `policies.txt`
- Simple menu-driven console interface

## Technologies Used

- C++ programming language
- File handling (fstream)
- Struct and functions
- Console application

## How to Run

1. Open terminal in the project folder.
2. Compile the code: `g++ main.cpp -o licms`
3. Run the program (PowerShell): `.\licms`

## Project Files

- `main.cpp` – source code
- `policies.txt` – data file (created automatically after adding first policy)
